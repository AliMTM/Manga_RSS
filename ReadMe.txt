Manga Scraper with Chat GPT-3
This project is built with the help of Chat GPT-3 and allows you to scrape all the manga released daily from "https://manganato.com/" website. The scraper collects only those manga that have a rating of 4.8 or higher and at least one chapter.

Once the manga is collected, the scraper generates an HTML page to represent the collected manga. This page provides a convenient way to browse and access your favorite manga all in one place.

To use this scraper, simply run the script and the scraper will automatically gather and sort the manga based on the specified criteria. With this Manga Scraper, you can easily stay up-to-date with the latest and most popular manga releases!
